How to submit your entry

  1. Fill in the form and click "Add to dataset"
  2. Review the diff below
  3. Download the updated publications.json or the .patch file
  4. Submit a Pull Request on GitHub:
    • Fork the repository
    • Option A: Replace assets/data/publications.json with your downloaded file
    • Option B: Apply the downloaded patch: 
      cd how-to-build-digital-humans
git apply entry.patch
    • Open a Pull Request

Automated checks

When you open a PR that modifies publications.json, a CI check will automatically validate your entry (schema, venue, links, duplicates) and post the results as a PR comment.

Building locally

After applying your changes to publications.json, run the following to fetch thumbnails, abstracts, and rebuild the site: 

./deploy.sh
